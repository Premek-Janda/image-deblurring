# POVa Project on image deblurring


## Setup
```bash
pip install --user virtualenv
virtualenv -p python3 image-deblurring
```

on linux:
```bash
source image-deblurring/venv/Scripts/activate
```

on win:
```bash
.\image-deblurring\venv\Scripts\activate
```

```bash
pip install -r requirements.txt
```
